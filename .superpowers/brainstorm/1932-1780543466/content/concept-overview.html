Interaction States

Auto-Play

Each icon's preview plays ~5 seconds, then active moves to next icon in sequence. Box around active icon glows.

Hover

Hovering an icon switches preview to that icon's video. On unhover, returns to whichever icon was active before hover (saved state).

Click (Lock)

Clicking locks the active icon. Video plays fully (loops?) until clicked again to unlock. Returns to auto-rotate.
