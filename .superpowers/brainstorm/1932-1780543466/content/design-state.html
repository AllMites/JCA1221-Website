Design: State Machine

3-tier priority: Click > Hover > Auto

State Variables
activeIndex: number — currently executing icon
lockedIndex: number | null — clicked/locked icon
hoveredIndex: number | null — hovered icon
savedIndex: number | null — pre-hover activeIndex
videoState: 'playing' | 'paused' | 'ended'
Resolution Order (what video plays)
1. Locked? → show lockedIndex video, loop/play full
2. Hovered? → show hoveredIndex video, save current activeIndex
3. Auto → show activeIndex video, 5s per icon

On unhover: restore savedIndex → activeIndex
On unlock: lockedIndex = null, resume from current icon
On click new icon: lockedIndex = icon, video play full
On click locked icon: unlock it
Auto-Rotation Timer
  • Each active icon plays 5 seconds, then advances
  • Timer pauses during hover or lock
  • On hover leave (not locked): timer resumes from where it left off
  • On unlock: timer starts fresh for current icon
  • Video holds last frame when timer expires (no crossfade needed)
