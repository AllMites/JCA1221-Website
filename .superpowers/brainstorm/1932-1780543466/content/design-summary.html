Full Design Summary

Technology Carousel Section — final design for approval

What We're Building

New TechnologyCarouselSection sits above ProcessFlowSection on the /technology page. 5 icons along the "waste to resource" eyebrow line, each with a video/GIF preview. Active icon glows (box shadow), auto-rotates every 5 seconds, hover temporarily switches, click locks.

State Machine

Priority: Click > Hover > Auto
Locked stays locked until unclicked. Hover saves previous state. Auto rotates 5s per icon, holds last frame.

Approach

A: Single video element
src swaps on icon change. No framer-motion crossfade. Native video API.

Glow (No Orb)

Active: cyan box-shadow + border
Hovered: slightly dimmer cyan
Locked: amber box-shadow
Inactive: dimmed, no glow
Thin connecting lines. No orb.

Files

TechnologyCarouselSection.tsx — new component
TechnologyView.tsx — add above ProcessFlow
types.ts — TechnologyStep interface
data.json — videoSrc per step (placeholders)

Responsive

Desktop: icon row + 16:9 video (max-w 800px) + step description below.
Mobile: icons horizontal scroll, video full-width, description stacked below.

Edge Cases

  • Video fails to load → show static placeholder with icon
  • Autoplay blocked by browser → show play button overlay, start on user gesture
  • prefers-reduced-motion → skip auto-rotation, manual icon selection only
  • No videoSrc provided yet → show icon + label placeholder card
  • Rapid hover across icons → debounce 150ms before switching video
